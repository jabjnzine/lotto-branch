# ระบบหวย — คู่มือการใช้งาน (User Guide)

## ภาพรวม

ระบบ Back Office สำหรับจัดการหวย ครบวงจรตั้งแต่คีย์บิล จัดการเลขอั้น บันทึกผลหวย คำนวณถูก-ผิด ดูรายงานสรุปรายได้

---

## 1. หน้าเข้าสู่ระบบ (Login)

**Path:** `/login`

| ฟังก์ชัน | คำอธิบาย |
|----------|----------|
| เข้าสู่ระบบ | กรอกอีเมลและรหัสผ่านเพื่อเข้าใช้งานระบบ |
| ตรวจสอบสิทธิ์ | ระบบตรวจสอบ JWT Token + Refresh Token (อายุ 7 วัน) |

---

## 2. หน้า Dashboard

**Path:** `/dashboard`

| ฟังก์ชัน | คำอธิบาย |
|----------|----------|
| Quick Links | ปุ่มลัดไปหน้า คีย์หวย, เลขอั้น, ผลหวย, รายได้ |
| KPI สรุปวันนี้ | แสดงจำนวนบิล, ยอดรับ, ยอดจ่าย, กำไร/ขาดทุน ของวันนี้ |
| งวดวันนี้ | รายการงวดที่เปิดรับวันนี้ พร้อม Countdown นับถอยหลัง |
| แสดง 0 เมื่อไม่มีข้อมูล | ทุก KPI แสดง 0 แทนที่จะซ่อน |

---

## 3. หน้าคีย์หวย (Bet)

**Path:** `/bet`

| ฟังก์ชัน | คำอธิบาย |
|----------|----------|
| เลือกประเภทหวย | สลับระหว่างหวยรัฐบาล, หวยลาว, หวยลาวพัฒนา |
| เลือกประเภทการแทง | 3 ตัวบน/โต้ด/หน้า/หลัง, 2 ตัวบน/ล่าง, วิ่งบน/ล่าง, 4 ตัวบน, 5 ตัวบน |
| แสดงอัตราจ่าย | อัตราจ่ายแสดงในช่องมุมขวาของแต่ละประเภท |
| เลขกลับ | สำหรับ 2-3 ตัว — กดเพื่อแทงเลขกลับโดยอัตโนมัติ |
| ใส่วงเงิน | ระบุจำนวนเงินต่อรายการ |
| ตรวจจับเลขอั้น | ถ้าเลขถูกปิดหรือจำกัดวงเงิน ระบบจะแจ้งเตือนทันที |
| เพิ่มรายการ | เพิ่มเข้ารายการแทงด้านบน |
| ลบรายการ | กดไอคอนถังขยะเพื่อลบรายการ |
| สรุปยอดรวม | แสดงยอดรวมทั้งสิ้นด้านขวา |
| บันทึกบิล | **F2** — บันทึกบิลและแสดงใบเสร็จ |
| ล้างข้อมูล | **F3** — ล้างทุกรายการและฟอร์ม |
| พิมพ์ใบเสร็จ | **F4** — บันทึกและพิมพ์ใบเสร็จ |
| ส่งออก Excel | **F5** — ส่งออกรายการเป็นไฟล์ Excel |
| ระบุชื่อลูกค้า | กรอกชื่อและเบอร์โทร (ไม่บังคับ) |
| หมายเหตุ | ระบุหมายเหตุเพิ่มเติม (ไม่บังคับ) |
| Countdown ปิดรับ | แสดงเวลาที่เหลืองวดนี้ |

## 4. หน้าเลขอั้น (Restrictions)

**Path:** `/restrictions`

| ฟังก์ชัน | คำอธิบาย |
|----------|----------|
| เลือกประเภทหวย | เลือกหวยที่ต้องการจัดการเลขอั้น |
| เลือกประเภทการแทง | เลือกประเภทที่ต้องการอั้น |
| 3 รูปแบบการอั้น | **ปิดรับ** — ไม่รับเลขนี้เลย, **จำกัดวงเงิน** — รับได้ไม่เกินที่กำหนด, **จ่ายครึ่ง** — ถูกลดครึ่ง |
| วงเงินสูงสุด | ระบุวงเงินสูงสุดสำหรับแบบจำกัดวงเงิน |
| เพิ่มเลขอั้น | กดเพื่อเพิ่มเลขอั้น |
| รายการเลขอั้น | แสดงเลขอั้นทั้งหมดของงวดปัจจุบัน |
| ลบเลขอั้น | ยืนยันลบผ่าน Confirm Dialog |
| แสดงจำนวนรายการ | แสดงจำนวนเลขอั้นทั้งหมดที่ตั้งไว้ |

---

## 5. หน้าผลหวย (Results)

**Path:** `/results`

| ฟังก์ชัน | คำอธิบาย |
|----------|----------|
| เลือกประเภทหวย | เลือกหวยที่ต้องการบันทึกผล |
| ดึงผลอัตโนมัติ (ไทย) | กด "ดึงผลหวยรัฐบาลอัตโนมัติ" — ดึงจาก API ภายนอก |
| ดึงผลอัตโนมัติ (ลาว) | กด "ดึงผลหวยลาวอัตโนมัติ" — ดึงจากเว็บหวยลาว |
| บันทึกผลแบบ Manual | กรอกผลเอง: รางวัลที่ 1, เลขหน้า 3 ตัว, เลขท้าย 3 ตัว, เลขท้าย 2 ตัว |
| LAO 5 ชุด 2 ตัว | กรอกผลเป็น 5 ช่อง 2 ตัว |
| บันทึกผล | กดบันทึกผล และระบบจะคำนวณถูก-ผิดให้อัตโนมัติ |
| คำนวณถูก-ผิด | กดเพื่อคำนวณใหม่ (กรณีต้องการแก้ไขผล) |
| สถานะ Official | แสดงว่าบันทึกแบบ Manual หรือ Official |

## 6. หน้ารายงาน (Reports)

**Path:** `/reports`

| ฟังก์ชัน | คำอธิบาย |
|----------|----------|
| ทั้งหมด | มุมมองรวมทุกประเภทหวยของวันนี้ |
| เลือกประเภทหวย | ดูเฉพาะหวยที่ต้องการ |
| เลือกงวด | ดูรายงานของงวดนั้นๆ |
| KPI ภาพรวม | วันที่, จำนวนบิลทั้งหมด, ยอดรับรวมทั้งวัน |
| ตารางแยกตามหวย | การ์ดแยกแต่ละประเภทหวย พร้อมสรุปยอดต่อประเภท |
| คอลัมน์แยก | เวลา, คนซื้อ, ประเภทหวย, เลข, ประเภทแทง, ยอด, สถานะ |
| Win/Loss Summary | สำหรับงวดที่ออกผลแล้ว — แสดงจำนวนถูก/ผิด, ยอดจ่าย, กำไร |
| ขยายดูรายละเอียด | กดที่แถวเพื่อดูรายละเอียดแต่ละรายการ |
| ยกเลิกบิล | กดยกเลิกบิลที่ยังรอผล (Pending) |
| Export Excel | ส่งออกเป็นไฟล์ Excel ทั้งงวด |
| Pagination | เลื่อนหน้าเมื่อมีรายการจำนวนมาก |
| ค้นหา | ค้นหาตามเลขหรือชื่อคนซื้อ (Real-time) |

## 7. หน้ารายได้ (Income)

**Path:** `/income`

| ฟังก์ชัน | คำอธิบาย |
|----------|----------|
| เลือกประเภทหวย | เลือกหวยที่ต้องการดูรายได้ |
| เลือกงวด | เลือกงวดที่ออกผลแล้ว |

| แยกตามประเภทการแทง | การ์ดแสดงยอดรับ/จ่าย/กำไร แยกตามประเภทการแทง |
| สีเขียว/แดง | กำไรแสดงสีเขียว, ขาดทุนแสดงสีแดง |

## 8. หน้าตั้งค่า (Settings)

**Path:** `/settings`

| ฟังก์ชัน | คำอธิบาย |
|----------|----------|
| เลือกประเภทหวย | เลือกหวยที่ต้องการตั้งค่า |
| เวลาออกผล | ตั้งเวลา draw_time |
| ปิดรับก่อนออกผล | ตั้งนาทีก่อนออกผลที่ต้องการปิดรับ |
| เปิด/ปิด ประเภทหวย | Toggle เปิด/ปิดการใช้งานประเภทหวย |
| อัตราจ่าย | ตั้งค่าอัตราจ่ายสำหรับแต่ละประเภทการแทง |
| วงเงินสูงสุดต่อเลข | ตั้งวงเงินสูงสุดที่รับได้ต่อหนึ่งเลข |
| สร้างงวด | กด "สร้างงวด" เพื่อ generate งวดล่วงหน้าอัตโนมัติ |

## 9. Shared Components (ใช้ร่วมกันทุกหน้า)

| Component | คำอธิบาย |
|-----------|----------|
| PageHeader | หัวข้อหน้า พร้อมคำอธิบาย |
| LoadingSpinner | แสดงระหว่างโหลดข้อมูล |
| StatCardSkeleton | แสดง Skeleton ระหว่างโหลด KPI |
| ConfirmDialog | Dialog ยืนยันการลบ/ยกเลิก |
| Pagination | ปุ่มเลื่อนหน้า ก่อนหน้า/ถัดไป |
| SearchInput | ช่องค้นหาพร้อมปุ่มล้าง |
| Receipt | ใบเสร็จสำหรับพิมพ์ |
| ErrorBoundary | แสดงข้อผิดพลาดเมื่อ Component Error พร้อมปุ่ม Refresh |
| Countdown | นับถอยหลังเวลาปิดรับ |
